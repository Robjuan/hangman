-------------------
SYSTEM REQUIREMENTS
-------------------
Python 2.6.4 or higher (not 3.x.x)  - http://www.python.org/download/releases/2.6.5/
pygame 1.9.1 or higher  - http://www.pygame.org/download.shtml

------------
INSTALLATION
------------
-> install python + pygame
-> extract all files into a folder
-> run game.py
